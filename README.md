# Food Menu Bootstrap
## Overview
This project is a responsive restaurant menu webpage for **Food, LLC**, built with **Bootstrap 3**. It features a mobile-friendly navbar that collapses into a hamburger menu and a clean layout for menu sections.
## Features
- Responsive Bootstrap 3 navbar with toggle menu on small screens  
- Navbar brand aligned left with `.navbar-brand` class  
- Dropdown menu that slides down and takes full width on mobile  
- Centered menu items and page heading using Bootstrap classes  
- Content section with enough height to enable scrolling  
- Internal CSS styling only (no inline styles)
## Usage
1. Clone the repository:  
   ```bash
   git clone https://github.com/software-engr/food-menu-bootstrap.git
````
2. Open `index.html` in your browser to view the menu page.
3. Resize your browser window to see the responsive navbar toggle in action.
## Technologies Used
* HTML5
* CSS3 (Internal styles)
* Bootstrap 3
* jQuery 3.x
